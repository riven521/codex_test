# 项目AGENT说明

## 项目概况
- 语言：Python。
- 目标：提供一个用于求解旅行商问题（TSP）的API和算法实现，包含数据结构、调度/路径规划接口、求解策略以及测试/示例。
- 预期模块：输入/输出接口、调度器、算法核心（如遗传、动态规划或启发式）、结果评估。

## 首要任务
1. 浏览仓库，找出已有的代码和测试，了解项目结构及依赖。
2. 针对TSP算法实现，写清楚输入格式、算法参数、返回结果、异常处理等契约。
3. 理清可运行的命令（如 `python -m`、`pytest`）、需要生成的配置或示例数据。

## 开发流程指引
- 变更代码前务必运行现有测试（`pytest` 或其他测试命令），提交前保证无失败。
- 优先维护具名函数、类、模块的文档化描述，尤其是与TSP参数/算法相关的部分。
- 对于新的依赖或第三方库，找出对应的许可信息并记录。

## 交付内容
- 新功能：需包含使用说明、示例数据、预期输出或对比结果。
- Bug 修复：说明触发路径、重现步骤、解决方案。
- 文档更新：保持 README 和 API 文档同步。

## 沟通要求
- 回复时使用中文（简体）。
- 如需外部参考或产出引用，注明具体来源路径。
